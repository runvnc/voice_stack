#### Cohere STT server - current state (afe1138):
- Native transformers>=5.4.0 API (`CohereAsrForConditionalGeneration` + `model.generate()`)
- Runs in `/opt/cohere-venv` (separate venv, avoids conflict with qwen-tts transformers pin)
- bfloat16 model loading
- **Encoder**: `torch.compile(mode='reduce-overhead')` + CUDA graphs
  - Pads raw audio before processor (not mel spectrogram) -> no attention_mask -> consistent CUDA graph shapes
  - `torch.compiler.cudagraph_mark_step_begin()` before each encoder call
  - 4 buckets: 150/300/500/1000 frames (~1.5/3/5/10s), all pre-warmed at startup
  - Warmup uses `(bucket-1)*160` samples to produce exactly `bucket` mel frames
- **Decoder**: each of 8 layers compiled with `torch.compile(dynamic=True)`
- Profiling: encoder/decoder/processor breakdown logged per request
- Profiling tools: `COHERE_PROFILE=1` (torch.profiler), `TORCH_LOGS=cudagraphs`
- Results before latest changes: encoder 2.6ms (CUDA graph), decoder ~30ms, total ~40ms

#### Mindroot integration (needs rebuild to activate):
- `with-mindroot` Dockerfile stage: `pip install mindroot` + plugins (mr_sip, ah_openrouter, mr_qwen3tts)
- All backend URLs point to localhost (no network round trip for STT/TTS/LLM)
- Credentials auto-generated at template creation, saved to `pod_state.json`
- Port 8010 added to RunPod template
- ENV baked in: `ANY_LLM_SERVER_URL=http://localhost:8000/v1`, `ANY_LLM_EXTRA_PARAMS`, `MR_QWEN3TTS_*`, `STT_PROVIDER=silero_cohere`, `COHERE_TRANSCRIBE_URL=http://localhost:8881`
